# Compliance Notes

This file documents the main compliance and Play Store considerations.

- Public privacy policy URL must be available before submission.
- Complete the Play Console Data Safety questionnaire accurately.
- Limit data collection to what is necessary and document retention.
- Provide contact/support email for Play Store listing.
- Keep permissions minimal and justify any sensitive permissions.

Operational:

- Ensure backups and restore procedures for PostgreSQL.
- Define a secrets management and rotation strategy.
- Add basic logging, rate limiting and alerting for production.
