# Roadmap

This roadmap outlines the planned phases toward a public release.

## Phase 1 — Product & Compliance
- Finalize MVP feature set
- Create privacy policy and Data Safety material
- Prepare Play Console assets and copy

## Phase 2 — Backend in production
- Publish staging and production APIs (HTTPS)
- Provision managed PostgreSQL with backups
- Add rate limiting, timeouts and basic observability

## Phase 3 — Mobile release
- Configure environment flags for release
- Point app to production API
- Produce signed `.aab` and validate on device

## Phase 4 — Pre-publication quality
- End-to-end device testing
- Degradation scenarios and latency tests

## Phase 5 — Play Console
- Upload release, submit to internal track
- Collect feedback and iterate
